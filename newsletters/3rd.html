You Renamed the Project?!

That's right! The project has gone through another renaming. Well, third time's the charm...

Say goodbye to "Andromedia", and hello to "ASTRAL DISC". The old name seemed a bit redundant when we had something like "ASTRAL DISC" already around, not to mention that "Andromedia" isn't THAT unique (it's already used in domain names, social media handles AND the name of an old movie...), as well as not having a clear enough meaning.

The website domain, and all mentions of the old project name have now been updated. We ask that from now on, please refer to the project as "ASTRAL DISC".

(Candleholder is still "Candleholder". Don't worry.)

Even More Progress...

We've been making lots of progress with the overall structure of the chapters, general timeline, progress documents... mostly internal stuff. Following the name change, we have also started polishing the chapter selection system we had in place, and it is now much more stable!

Chapter 3 and Chapter 4 are our main focus right now, with Chapter 5 also getting a few fun things here and there, like certain shops, polishing a certain segment in it, and many more changes and updates.

With this new reorganizing and rebranding of the project, we're hoping things run way more smoothly from now on!
